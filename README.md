Now with Rebar3 and Thumbs!

Following are a few of the commands you might run, look in the [Thumbs](.thumbs.yml) file for more.

```shell
$ rebar3 as prod compile
$ rebar3 as check xref
$ rebar3 as check dialyzer
$ rebar3 eunit
$ rebar3 edoc
```
